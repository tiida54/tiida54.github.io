ASP.NET Web Pages

ASP.NET 支持三種不同的開發模式：
Web Pages（Web 頁面）、MVC（Model View Controller 模型-視圖-控制器）、Web Forms（Web 窗體）

ASP.NET-Razor 標記

  • Razor 是一種將基於服務器的代碼添加到網頁中的標記語法
  • Razor 具有傳統ASP.NET 標記的功能，但更容易使用並且更容易學習
  • Razor 是一種服務器端標記語法，與ASP 和PHP 很像
  • Razor 支持C# 和Visual Basic 編程語言
