Adblock Plus - 10,000,000+ 位使用者

  • 超過 1 億部設備上使用，是世界上最受歡迎的廣告攔截插件
  • Adblock Plus for Google Chrome 會阻擋：橫幅廣告、YouTube 影片廣告、 Facebook 廣告、彈出式視窗、還有各種礙眼的廣告

FireShot - 1,789,710 位使用者

  • 捕捉網頁截圖，然後編輯，並將它們保存到PDF，JPEG，GIF，PNG或BMP，上傳，打印，在Photoshop中打開，複製到剪貼板或電子郵件

OneTab - 1,535,214 位使用者

  • 將所有分頁轉換成一個清單，需要再次訪問這些分頁時，可以單獨或全部還原
  • 節省高達95％的記憶體，並減輕分頁混亂現象

OneTab - Chrome 線上應用程式商店

Bookmark Manager - 904,946 位使用者

  • 將圖片和附註加入書籤，讓您更方便使用
  • 具有書籤搜尋功能，不用再為分類書籤而煩惱
  • 可以與親朋好友分享自己的書籤。

Bookmark Manager - Chrome 線上應用程式商店

Wappalyzer - 643,969 位使用者

  • 識別網站上的技術
  • 檢測分析工具，程式語言，ＪＳ框架，作業系統，網頁框架，內容管理系統，網頁伺服器，付款處理等技術

Wappalyzer - Identify technologies on websites

Dimensions - 74,005 位使用者

  • 設計人員測量屏幕尺寸的工具，測量一個網站上的元素之間的距離
  • 測量鼠標指針向上/向下和向左/向右的尺寸，直到碰到邊界

Dimensions - Chrome 線上應用程式商店

Web Developer Checklist - 46,002 位使用者

  • 網頁放上線之前的「查核清單」
  • 包含SEO、手機裝置、可用性、無障礙、環境整合、性能六大類
  • 還有詳細檢查清單Web Developer Checklist

Web Developer Checklist - Chrome 線上應用程式商店

Emmet Re:view - 41,179 位使用者

  • 快速簡單的在多個視窗中測試您的響應式設計
  • 以並排的方式在多個視圖中顯示您的響應式網頁，以便您可以快速測試網頁在不同分辨率和設備上的效果
  • 所有的視圖都完全同步：滾動，填寫表單域，單擊，懸停和拖動元素獲得即時反饋

Emmet Re:view - Chrome 線上應用程式商店
