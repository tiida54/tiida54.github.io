介紹5個設計常用的網站

成功不是將來才有的，而是決定去做的那一刻，持續累積而成。

Piskel

Piskel：一個免費的動畫圖片在線編輯器

  • 實時預覽：畫圖時，隨時查看動畫預覽。動態調整幀延遲。
  • 導出為GIF，PNG：支持多種導出模式－ GIF、PNG、ZIP
  • 私人畫廊：可以選擇向任何人公開或私人，私人只有你才能看見。
  • Google登錄：無需記住另一個密碼，只需使用您的Google帳戶登錄。
  • 開源GitHub：所有的代碼都是開源的，可以在GitHub上找到。
  • 離線版本：Windows，OSX和Linux的免費桌面和離線應用程序。

Color Hunt


Color Hunt：數千種不同的手工挑選調色板提供服務，並且每天都在增長

  • 每個調色板都由Color Hunt的用戶自己創建，其中最好的部分將在Color Hunt的主頁上顯示。
  • 所有調色板都可以自由使用。Facebook - Color Hunt

Iconfinder


Iconfinder：SVG，PNG，CSH和AI格式的搜索引擎

  • Iconfinder為數百萬創意專業人士提供高品質的圖標，世界上最受歡迎的圖標網站
  • 精選的免費圖標集，通過2,155,907個SVG圖標進行搜索

Loading.io

Loading.io：建立你的SVG / CSS / GIF / APNG的圖標

  • 用自己的圖案製作免費的Loading圖標
  • 製作具有特色的動畫圖標

Tiny PNG

Tiny PNG：智能PNG和JPEG壓縮，以質量和文件大小的完美平衡來優化圖像

  • 通過有選擇地減少圖像中的顏色數量，用更小的容量來存儲數據。
  • 文件大小減少了70％以上，優化圖像來節省加載時間，
