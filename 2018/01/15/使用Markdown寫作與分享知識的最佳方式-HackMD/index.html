使用Markdown寫作與分享知識的最佳方式

引言

MAC系列的使用者，可以使用 bear 來編寫 Markdown 格式文件，但windows系統卻沒有這樣的系統可以使用，所以今天來介紹一個，線上編輯Markdown的強大工具 HackHD

簡介

HackMD 是個跨平台的 Markdown 即時協作筆記
所以您可以在電腦、平板甚至是手機與其他人做筆記！
同時也可以在 首頁 透過 Facebook、Twitter、GitHub、Dropbox 登入

特色介紹

共同協作

檢視右上方有個小按鈕，您可以透過以下選項來更改權限：

隨意：任何人都可以更改這份筆記
可編輯：已登入使用者可以更改這份筆記
鎖定：只有擁有者可以更改這份筆記
私有：只有擁有者可以更改與檢視這份筆記

檢視模式

  • 檢視：只看到結果
  • 同時：同時看到兩邊
  • 編輯：只看到編輯器

上方工具列

工具列除了基本的文字粗體斜體，另外還提供了

  • 行內程式碼
  • 引用區塊
  • 列表清單
  • 待辦清單
  • 連結功能
  • 插入圖片
  • 插入表格
  • 插入分隔線

自動完成

提供完整的 Markdown 自動完成與提示

  • 表情符號：輸入 : 顯示提示
  • 程式碼區塊：輸入 ``` 加上一個字元 顯示提示 ```
  • 標頭：輸入 # 顯示提示
  • 參考：輸入 [] 顯示提示
  • 外部：輸入 {} 顯示提示
  • 圖片：輸入 ! 顯示提示

外部嵌入

提供多種格式的外部嵌入
Youtube、Vimeo、Gist、SlideShare、Speakerdeck、PDF

結語

除了以上的特色介紹， HackMD還擁有許多強大的功能，而且官方文件-HackHD功能介紹也說明得非常詳細，還提供中英日三種說明，快使用HackMD來編寫Markdown技術文檔吧！
